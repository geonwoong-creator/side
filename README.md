🚀 ITYS (I TOLD YOU SO) 내가 말했제
1. 서비스 정의
프로젝트명: ITYS (I TOLD YOU SO / 거봐, 내가 말했제)

핵심 컨셉: 친구들과 수익률 및 비중을 공유하며 투자 실력을 증명하는 프라이빗 소셜 포트폴리오 플랫폼.

슬로건: "입으로만 하지 말고, 숫자로 박제해라."

2. 핵심 운영 정책 (Privacy & Fun)
구분	항목	정책	비고
공개	수익률(%)	무조건 공개	실력 증명의 핵심 지표
공개	종목 비중(%)	무조건 공개	투자 전략 파악용
선택	보유 금액(₩)	사용자 선택	개인 프라이버시 보호
접근	그룹 시스템	초대 코드 전용	아는 사람들끼리의 폐쇄형 커뮤니티
3. 주요 기능 (Features)
다중 그룹 워크스페이스: 여러 개의 친구 모임에 참여하여 각기 다른 멤버들과 랭킹 경쟁.

간편 포트폴리오 관리: 종목명, 평단가, 수량 수동 입력 (최초 등록 시 즉시 시세 반영).

랭킹 및 비교 차트:

현재 수익률 기준 실시간 순위표.

시간 흐름에 따른 멤버별 수익률 변화를 보여주는 라인 차트.

"내가 말했제" (예측 박제): 특정 종목의 상승/하락을 선언하고 기한 종료 시 시스템이 자동 판정.

4. 기술 스택 (Technical Stack)
Frontend: Next.js (Web 기반, 반응형 설계)

Backend: FastAPI (Python)

Database & Auth: Supabase (PostgreSQL + Kakao/Google Auth)

Data Library: FinanceDataReader (국내 주식 시세 스크래핑)

Scheduler: APScheduler (1시간 주기 시세 갱신 및 일일 스냅샷 저장)
